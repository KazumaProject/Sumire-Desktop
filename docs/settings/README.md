# Sumire Settings README

`SumireSettings.exe` では、ライブ変換、候補ウィンドウ、`zenz` モデル、人名辞書ビルドをまとめて設定できます。設定はユーザーごとに保存され、主に `HKCU\Software\Sumire\Settings` を使います。

## 先に見る項目

- まず触る項目は `ライブ変換プレビュー`、`ひらがな補助ビュー`、`候補数` です。
- `zenz` を使う場合は `zenz 融合を有効にする` と `サービスプロセスを有効にする` を両方確認してください。
- 人名辞書を増やす場合は、`追加 / 更新` だけでは反映されません。`選択をビルド` または `すべてビルド` が必要です。

## 各設定項目

### ライブ変換プレビュー

- 入力中の読みから、未確定のまま第1候補を先に表示します。
- ON のときは、かなを打っている最中に `Preedit` がひらがなではなく変換候補になることがあります。
- OFF のときは、`Space` を押すまで通常の `Preedit` のままです。

### ひらがな補助ビュー

- ライブ変換プレビュー中に、元の読みを小さな補助ウィンドウで表示します。
- 変換候補が先に出ても、元のひらがなを見失いにくくするための表示です。
- ライブ変換を使わない場合は OFF でも問題ありません。

### 候補数

- 候補ウィンドウ 1 ページあたりの表示件数です。
- 既定値は `9` です。
- 実装上は `1` 以上、保存時は `50` までに丸められます。

### 表示言語

- `SumireSettings.exe` 自体の UI 言語です。
- 保存後、設定画面を開き直すと全ラベルに反映されます。
- IME の変換ロジックや入力モードはこの項目では変わりません。

### ローマ字マップ

- ローマ字からかなへの変換表ファイルを差し替えます。
- 何も指定しない場合は同梱の `romaji-hiragana.tsv` を使う想定です。
- 独自配列や補助キーを試すときに使います。

### zenz 融合を有効にする

- 辞書ベースの変換に加えて、`zenz` モデルによる候補補助を有効にします。
- OFF にすると、辞書ベースの変換だけを使います。
- モデルファイルが見つからない場合は警告が出ます。

### サービスプロセスを有効にする

- `SumireZenzService.exe` の起動・停止を許可します。
- ON のときは必要に応じて起動し、OFF のときは停止を試みます。
- `zenz` を使うなら ON を推奨します。

### モデル preset / モデルファイル / 既定モデル repo

- どの GGUF モデルを使うかを決める項目です。
- `preset` は既定パスと既定 repo のセットを選ぶための補助です。
- `モデルファイル` に実ファイルの場所、`repo` にダウンロード元の URL を入れます。
- インストーラーでモデルを取得しなかった場合は、ここでパスを合わせてください。

## 人名辞書ビルド

独自の人名辞書は、テキストをそのまま読むのではなく、設定画面から `.bin` にビルドして使います。

### 手順

1. `名前` を入れます。
2. `元 TXT` に辞書テキストのパスを指定します。
3. `実行時に有効` を必要に応じて ON/OFF します。
4. `追加 / 更新` を押してプロファイル登録します。
5. `選択をビルド` または `すべてビルド` を押します。
6. 完了後、IME プロセスを再起動してビルド済み辞書を再読込します。

### 元 TXT の形式

- UTF-8 テキストを使います。
- 1 行 5 列のタブ区切りです。
- 列順は `reading<TAB>left_id<TAB>right_id<TAB>cost<TAB>surface` です。
- `#` で始まる行と空行は無視されます。

例:

```text
すみれ	1000	1000	8000	菫
かずま	1000	1000	8000	一真
```

### 出力先

- 既定ではインストール先配下の `dictionaries\names\build\*.bin` に出力されます。
- `元 TXT` を変えると、既存のビルド済みパスはクリアされて再ビルドが必要になります。

## 保存と反映

- 通常の設定変更は `保存` で反映されます。
- 表示言語の変更は `保存` 後に設定画面を再起動してください。
- 辞書ビルド結果の反映には IME プロセスの再起動が必要です。

## 困ったとき

- モデル未検出の警告が出る: `モデルファイル` を見直すか、インストーラーでモデルを取得してください。
- サービスが起動しない: `SumireZenzService.exe` がインストール先にあるか確認してください。
- 辞書が効かない: `追加 / 更新` の後に `ビルド` を行い、IME を再起動してください。
